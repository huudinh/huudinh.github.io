            Variable swap
            Swapping variable is to exchange the values of two variable 
            so that at the end, one variable contains the value of the other

            For example:

            let a = 5;
            let b = 6;
            console.log(a, b); // Result: 5 6

            

            console.log(a, b); // Result: 6 5
